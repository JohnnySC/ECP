# ECP  v2.0
Вторая итерация проекта ECP (первая находится в ветке `olddev`). Проект представляет собой чат-бота, выполняющего различные команды.

Написан по принципам ООП, SOLID, DRY, KISS и др.

Стек: CoreMVVM
